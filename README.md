## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AnthonyClermont/iventis-application.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run project
   ```sh
    npm run dev
   ```

## Run Tests
